
# 冷刃博弈 — 核心规则文档

> 以「距离管控」为核心的回合制中国传统冷兵器对战体系
> 用极简规则还原「一寸长一寸强，一寸短一寸险」的冷兵器博弈本质

---

## 〇、项目概述

### 设计理念

所有规则围绕一个核心：**双方的相对距离决定一切**。

- 兵器的伤害加成/惩罚全部绑定距离区间。脱离优势距离，再强的操作也打不出效果
- 用「选卡」替代「搓招+手速」，把冷兵器博弈简化为**控距 + 预判**的核心循环
- 高收益必带高风险，每张牌都有克制关系和失误惩罚，没有无敌操作

### 项目目标

| 维度 | 目标 |
|------|------|
| 开发验证 | 用最低成本跑通「距离核心博弈闭环」，验证平衡性和游玩乐趣 |
| 玩家体验 | 比的不是连招和手速，而是控距、预判、攻防节奏把控 |
| 长线差异化 | 建立以距离博弈为核心的玩法壁垒 |

### 技术栈

JavaScript (ES Modules) + Vite 5.4 构建，无框架依赖。纯前端单页应用。

```bash
npm install
npm run dev      # 开发服务 localhost:5173
npm run build    # 生产构建
```

## 一、术语表

| 术语 | 定义 |
|------|------|
| **距离** | 双方之间的共享相对状态，整数 0-3 |
| **僵直** | 下一回合禁止使用所有攻击类卡牌（重击/轻击/擒拿），仅可移动+防守 |
| **优势区间** | 该兵器触发攻击加成的距离范围 |
| **劣势区间** | 该兵器触发攻击惩罚的距离范围（伤害降低，但卡牌仍可使用） |
| **处决** | 架势值被削减至 0 点时被动触发，扣除 5 点气血，架势值恢复满 |
| **身法打断** | 移动中被攻击命中扣血，移动被取消 |

## 二、角色属性

双方共用同一套数值规则：

| 属性 | 数值 | 规则 |
|------|------|------|
| 气血 | 10 点 | 归零即败。仅被攻击和处决扣除，不回复 |
| 体力 | 4 点（上限） | 每回合结束恢复 1 点（扎马额外 +1 = 恢复 2）。**身法层消耗体力**（冲步/撤步/闪避） |
| 架势 | 5/5 点 | 初始满值，被命中会减少。降至 0 点 → 触发处决（扣5 气血），然后恢复满值 |

> **重要**：闪避归为身法层，消耗 2 体力（与冲步/撤步相同）。闪避可规避重击/轻击，但**擒拿可穿透闪避**，优势区轻击可打断闪避。其他攻防卡（卸力、重击、轻击、格挡、擒拿）不消耗体力。体力用于身法层（移动 + 闪避）。

## 三、距离系统

### 3.1 四个距离区间

**对局初始距离：2（中距区）**

```
 0-贴身  ←→  1-近战  ←→  2-中距  ←→  3-远距
 ├─ 短刀优势 ─┤              ├─ 长枪优势 ─┤
 │双刺│
              ├── 剑 优势 ──┤
              ├── 棍 优势 ─────────────┤
                             │ 大刀优势 │
```

| 距离值 | 区间名 | 短刀 | 长枪 | 剑 | 棍 | 大刀 | 双刺 |
|--------|--------|------|------|-----|-----|------|------|
| 0 | 贴身区 | **优势** | **劣势** | **劣势** | **劣势** | **劣势** | **优势** |
| 1 | 近战区 | **优势** | 普通 | **优势** | **优势** | 普通 | 普通 |
| 2 | 中距区 | 普通 | **优势** | **优势** | **优势** | **优势** | **劣势** |
| 3 | 远距区 | **劣势** | **优势** | **劣势** | **优势** | 普通 | **劣势** |

### 3.2 身法系统

每回合双方各选 1 张身法卡，**效果叠加后结算**：

| 卡牌 | 效果 | 消耗 |
|------|------|------|
| 冲步 | 间距 -1 | 2 体力 |
| 撤步 | 间距 +1 | 2 体力 |
| 扎马 | 不变 | 0 体力（回合结束额外恢复 +1 体力） |

- 双方效果叠加：「双方都冲步」→ 间距 -2；「一冲步一撤步」→ 间距不变
- 边界锁定：结算后 clamp 到 [0, 3]

### 3.3 距离对伤害的影响

- **优势区**：武器获得攻击加成（+伤害）
- **劣势区**：武器攻击受到惩罚（-伤害），惩罚严重时伤害降为 0（等于空招）
- **所有卡牌始终可用**，不会因距离被禁用——但在劣势区攻击效果极差
- 劣势区卡牌显示为"虚线框" + "⚠ 距离不佳"

## 四、卡牌系统

每回合必须出 **1 张身法卡 + 1 张攻防卡**，不能空过。

### 4.1 攻防卡一览

| 卡牌 | 类型 | 基础伤害 | 效果 |
|------|------|----------|------|
| **💨 闪避** | 身法 | — | 消耗 2 体力，规避重击/轻击伤害，对手 -1 架势。**擒拿可穿透闪避**。短刀/双刺闪避仅耜 1 体力 |
| **🤺 卸力** | 防守 | — | 反制重击（对手受 2 伤-2 架势+僵直）+ 识破擒拿（对手-2 架势）；遇其他牌受惩罚 |
| **⚡ 重击** | 攻击 | 3 | 重击，穿透格挡（减 1 伤），可被卸力反制 |
| **🎯 轻击** | 攻击 | 1 | 快攻 + 对手 -1 架势，可被格挡完全抵消 |
| **🛡️ 格挡** | 防守 | — | 完全抵消轻击；减免重击 1 点伤害 |
| **🌀 擒拿** | 攻击 | 0 | 克格挡和闪避：穿透闪避，命中格挡时对手 -3 架势。被卸力识破 |

#### 卸力结果一览

| 对手出牌 | 卸力方结果 |
|----------|------------|
| **重击** | **反制成功**：对手受 2 伤 - 2 架势 + 僵直 |
| **擒拿** | **识破成功**：对手 -2 架势 |
| 轻击 | 吃轻击全额伤害，自身 -1 架势 |
| 闪避/格挡 | 自身 -1 架势（浪费） |
| 卸力（对碰） | 双方各 -2 架势 |

### 4.2 完整 6×6 克制矩阵

行=我方出牌，列=对手出牌:

| 我方 ＼ 对手 | 闪避 | 卸力 | 重击 | 轻击 | 格挡 | 擒拿 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **闪避** | 空过 | 空过 | 闪开！对手-1架势 | ★见下方 | 空过 | 闪避落空！擒拿穿透 |
| **卸力** | 自身-1架势 | 各-2架势 | **反制2伤-2架势+僵直** | 受轻击伤-1架势 | 自身-1架势 | **识破！对手-2架势** |
| **重击** | 落空-1架势 | **被反制2伤-2架势+僵直** | 互砍各受伤 | 命中3伤-1架势 | 破防减1伤后命中 | 命中3伤-1架势 |
| **轻击** | ★见下方 | 穿透1伤-1架势 | 被克3伤-1架势 | 互刺各受伤 | 被挡完全抵消 | 命中1伤-1架势 |
| **格挡** | 空过 | 空过(对手-1架势) | 受减伤后伤害-1架势 | 格挡完全抵消 | 空过 | 被骗自身-3架势 |
| **擒拿** | 穿透闪避！ | 被识破：自身-2架势 | 被砍3伤-1架势 | 被刺1伤-1架势 | 骗到对手-3架势 | 空过 |

> **★ 优势区轻击破闪避**：当轻击方处于自身兵器优势区时，轻击可**无视闪避**直接命中。  
> **★ 擒拿穿透闪避**：擒拿无视闪避，闪避方闪避落空（但攻防卡照常生效）。

### 4.3 核心克制链

```
⚔️剑 → 克 → 🔱长枪、🌀擒抿、🛡️格挡(破防)  
🎯轻击 → 克 → 🤺卸力、🌀擒拿  
🤺卸力 → 克 → ⚡重击 (反制2伤+僵直)、🌀擒拿 (识破-2架势)  
🛡️格挡 → 克 → 🎯轻击 (完全抵消)  
🌀擒抿 → 克 → 🛡️格挡 (骗出-3架势)、💨闪避 (穿透闪避)  
💨闪避 → 克 → ⚡重击、🎯轻击(非优势区)，⚠无法闪避擒抿  
```

## 五、兵器系统

共 6 把兵器。所有加成/惩罚由距离触发。**劣势区攻击架势效果减半（向下取整）。**

### 5.1 短刀 🗡️（近身刺客）

- **优势区间：0-1** | **劣势区间：2-3**
- 优势区：擒拿-4架势，轻击破闪避，**闪避反击1伤**
- 劣势区：重击-3伤（→0）
- **特殊规则**：优势区轻击可直接打断闪避；优势区闪避成功自动反击对手1伤；**闪避仅耗 1 体力**

### 5.2 长枪 🔱（远距控场型）

- **优势区间：2-3** | **劣势区间：0（贴身）**
- 优势区：重击+2伤（→5），重击命中额外-1架势，**格挡弹枪推1距**
- 劣势区：重击-3伤（→0）
- **特殊规则**：重击命中后对手额外-1架势；优势区格挡成功→推开对手1格（弹枪效果）

### 5.3 剑 ⚔️（防反兼备型）

- **优势区间：1-2** | **劣势区间：0, 3**
- 优势区：卸力不僵直/自身+2架势，**完美格挡（重击完全免伤）**
- 劣势区：距离0→重击-2伤；距离3→重击-3伤
- **特殊规则**：卸力成功不造成僵直，改为自身架势+2；优势区格挡重击完全免伤

### 5.4 棍 🏏（控场压制型）

- **优势区间：1-3（最宽）** | **劣势区间：0（贴身）**
- 优势区：擒拿-4架势（基砀3+1），**重击额外-2架势**，**格挡震退-1架势**
- 劣势区：重击-3伤（→0）
- **特殊规则**：擒拿命中格挡时额外推间距+1；优势区重击弱伤高控(3伤-3架势)；优势区格挡成功给攻击方-1架势

### 5.5 大刀 🪓（重击爆发型）

- **优势区间：仅2（最窄）** | **劣势区间：0（贴身）**
- 优势区：重击+3伤（→6，全场最高），格挡减免重击2伤（而非1伤）
- 劣势区：重击-3伤（→0）
- **特殊规则**：重击命中后推间距+1（一刀劈退对手，但也离开自己优势区）

### 5.6 双刺 🥢（极近贴身型）

- **优势区间：仅0** | **劣势区间：2-3**
- 优势区：轻击+1伤（→2），擒拿-4架势
- 劣势区：重击-3伤（→0）
- **特殊规则**：闪避成功时对手-2架势（基砀1+1）；优势区轻击破闪避；**闪避仅耜 1 体力**

### 六把兵器总览

| 兵器 | 优势区 | 劣势区 | 核心优势 | 核心代价 |
|------|--------|--------|----------|----------|
| 🗡️ 短刀 | 0-1 | 2-3 | 轻击破闪避，闪避反击1伤 | 远距重击无效 |
| 🔱 长枪 | 2-3 | 0 | 重击5伤-2架势，格挡弹枪推距 | 贴身几乎无伤 |
| ⚔️ 剑 | 1-2 | 0, 3 | 卸力不僵直/回架势，完美格挡 | 无极端爆发 |
| 🏑 棍 | 1-3 | 0 | 擒拿-3架势+推人，重击-2架势，格挡震退 | 贴身无力 |
| 🪓 大刀 | 2 | 0 | 重击6伤+推距，格挡厚 | 仅1格优势 |
| 🥢 双刺 | 0 | 2-3 | 轻击+1伤，闪避-2架势 | 仅1格优势 |

## 六、回合流程

```
┌─────────────────────────────────────────────────────────┐
│ ① 信息同步：展示距离、双方属性、兵器优劣势               │
│ ② 玩家出牌：选 1 张身法卡 + 1 张攻防卡                   │
│ ③ AI 出牌：根据难度策略暗选                               │
│ ④ 身法结算：叠加计算新间距                                │
│ ⑤ 攻防结算：按克制矩阵结算伤害/架势                      │
│ ⑥ 身法打断判定：移动中受伤→移动取消                      │
│ ⑦ 状态结算：处决检查、体力恢复、回合结束                  │
└─────────────────────────────────────────────────────────┘
```

### 身法打断机制

- **触发条件**：本回合选了冲步/撤步的一方，在攻防结算中受到了HP伤害
- **效果**：该方的移动被取消，距离回退（另一方移动照常生效）
- **扎马不受影响**

## 七、AI 系统

8级难度从菜鸟到绝世高手：

| 难度 | 名称 | 特点 |
|------|------|------|
| 1 | 菜鸟 | 偏向随机，略偏好优势区（60%倾向） |
| 2 | 学徒 | 40% 概率反制，基础距离感知 |
| 3 | 弟子 | 55% 反制 + 僵直惩罚利用，基础闪避判断 |
| 4 | 镖师 | 65% 反制 + 连招链（擒拿→重击），资源管理意识 |
| 5 | 武师 | 75% 反制 + 距离预判 + 体力优化，锁控优势区间 |
| 6 | 高手 | 85% 反制 + 深度分析 + 架势压制，处决意识 |
| 7 | 宗师 | 反模式打法，诱导出牌 + 反向预测，陷阱策略 |
| 8 | 绝世 | 二阶预测 + 纳什混合策略，几乎无破绽 |

> 高级AI不作弊：基于历史出牌模式预测，玩家改变策略可让AI判断失误。

### 极端对局（>80%胜率差）

| 对局 | 优势方胜率 |
|------|-----------|
| 大刀 vs 双刺 | 86.6% |
| 长枪 vs 双刺 | 86.1% |
| 棍 vs 短刀 | 85.5% |
| 剑 vs 短刀 | 83.4% |

### 平衡分析

- **剑和棍**是最全面的武器，宽优势区让它们在大多数距离都能输出
- **短刀和双刺**过度依赖贴身，被远程武器轻松拉开距离压制
- **大刀**爆发极高但优势区太窄（仅距离2），命中后还自动推远离开优势区
- **高平局率**（35.1%）源于50回合上限和部分武器的防守僵持

## 九、游戏模式

### 散打（自由对战）

选择双方武器和AI难度，进行单局对战。

### 江湖行（闯关模式）

使用随机分配的武器，依次挑战多个NPC对手。每轮使用不同武器组合。

### 模拟系统

内置AI对战模拟器，可配置：
- 双方AI等级
- 每组对局数（50/100/500/1000）
- 自动生成6×6胜率矩阵
